
# Анкета с отправкой в Telegram (Vercel)

Форма валидирует все поля (`required`), генерирует **новый трек‑код** при каждом отправлении,
показывает его в модальном окне и отправляет всю заявку в Telegram через серверлес‑функцию.

## Как развернуть на Vercel без токенов в коде

1. Залейте репозиторий на GitHub.
2. Подключите его в Vercel.
3. В проекте Vercel откройте **Settings → Environment Variables** и добавьте:
   - `TELEGRAM_BOT_TOKEN` — токен вашего бота
   - `TELEGRAM_CHAT_ID` — ID группы/чата, куда слать сообщения
4. Задеплойте. Клиент отправляет POST на `/api/submit`, функция читает секреты из ENV и шлёт в Telegram.
5. Проверьте страницу `/form.html`.

> Никаких секретов в исходниках нет; в локальном превью функция вернёт ошибку 500 до тех пор,
> пока не будут заданы переменные окружения.
